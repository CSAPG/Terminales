📖 1. Rappel — Mole et quantité de matière

Notations et unités

La quantité de matière se note et s'exprime en de symbole : .


Si le nombre d'entités élémentaires d'un échantillon est N (sans unité), la quantité de matière contenue dans un échantillon est :

avec NA : constante d'Avogadro (NA = 6,02 × 10²³ mol⁻¹)
⚠️ Remarque : La quantité de matière ne se mesure pas directement — on passe par la masse ou le volume.

✏️ Application — Nombre d'entités

Un échantillon contient n = 0,50 mol de molécules d'eau H₂O. Combien contient-il de molécules ?
Rappel : NA = 6,02 × 10²³ mol⁻¹

N = molécules
N = n × NA = 0,50 × 6,02 × 10²³
N = 3,01 × 10²³ molécules
⚖️ 2. Espèce chimique pure — solide ou liquide

a. Masse molaire

La masse molaire d'une espèce chimique, notée , est la masse d'une mole de cette espèce. Elle s'exprime en .

💡 Rappel : Pour les molécules, M = somme des masses molaires atomiques de tous les atomes. Pour les ions, on néglige la masse des électrons.

✏️ Application — Masse molaire de l'eau H₂O

Données : M(H) = 1,0 g·mol⁻¹  |  M(O) = 16,0 g·mol⁻¹

M(H₂O) = g·mol⁻¹
M(H₂O) = 2 × M(H) + M(O) = 2 × 1,0 + 16,0
M(H₂O) = 18,0 g·mol⁻¹

b. Détermination par la masse

Dans le système international, l'unité de masse est le de symbole . En chimie, on utilise plutôt le .


La relation entre la masse m (g), la quantité de matière n (mol) et la masse molaire M (g·mol⁻¹) est :

soit m = n × M  |  M = m / n

✏️ Application — Quantité de matière d'un solide

On pèse 11,7 g de chlorure de sodium NaCl.
Données : M(Na) = 23,0 g·mol⁻¹  |  M(Cl) = 35,5 g·mol⁻¹

n(NaCl) = mol
M(NaCl) = 23,0 + 35,5 = 58,5 g·mol⁻¹
n = m / M = 11,7 / 58,5
n(NaCl) = 0,200 mol

Vérification dimensionnelle : g ÷ (g·mol⁻¹) = mol ✓

🧪 3. Espèce chimique en solution
Une solution est obtenue par dissolution d'une espèce chimique, le soluté, dans un liquide, le solvant (l'eau le plus souvent).

Concentrations

La concentration massique (ou titre massique) Cm en g·L⁻¹ :

La concentration molaire c en mol·L⁻¹, directement liée à n et au volume de solution Vs :

soit n = c × Vs  |  Vs = n / c

✏️ Application — Concentration molaire d'une solution

On dissout 8,00 g de NaOH (M = 40,0 g·mol⁻¹) dans de l'eau pour obtenir 500 mL de solution.
Calculer la concentration molaire c.

c(NaOH) = mol·L⁻¹
n(NaOH) = m / M = 8,00 / 40,0 = 0,200 mol
Vs = 500 mL = 0,500 L
c = n / Vs = 0,200 / 0,500
c(NaOH) = 0,400 mol·L⁻¹

Dissolution — masse à peser

Pour préparer un volume V de solution de concentration c avec un soluté de masse molaire M :

Rappel opératoire : peser → verser dans la fiole jaugée → compléter avec le solvant jusqu'au trait de jauge → agiter.

✏️ Application — Masse de glucose à peser

On veut préparer 250 mL d'une solution de glucose C₆H₁₂O₆ à c = 0,100 mol·L⁻¹.
Données : M(glucose) = 180 g·mol⁻¹

m = g
V = 250 mL = 0,250 L
m = c × V × M = 0,100 × 0,250 × 180
m = 4,50 g

On pèse 4,50 g de glucose, on les introduit dans une fiole jaugée de 250 mL, on complète avec de l'eau distillée jusqu'au trait de jauge.

c1, V1 : solution mère  |  c2, V2 : solution fille

Volume à prélever (solution mère) :

Facteur de dilution : F = c1/c2 = V2/V1

✏️ Application — Volume à prélever pour une dilution

On veut préparer V₂ = 200 mL d'une solution de HCl à c₂ = 0,050 mol·L⁻¹ à partir d'une solution mère à c₁ = 1,00 mol·L⁻¹.
Quel volume V₁ faut-il prélever ?

V₁ = mL
c₁ × V₁ = c₂ × V₂
V₁ = (c₂ × V₂) / c₁ = (0,050 × 200) / 1,00
V₁ = 10,0 mL

Facteur de dilution F = c₁/c₂ = 1,00/0,050 = 20 → on divise le volume par 20 : 200/20 = 10 mL ✓
On utilise une pipette jaugée de 10 mL et une fiole jaugée de 200 mL.

📐 4. Autres formules utiles

Passage % massique → concentration molaire

Le pourcentage massique %M d'un soluté A est défini par :

%M = [m(A) / mtotale] × 100
m(A) : masse du soluté (g)  |  mtotale : masse totale de la solution (g)

On connaît %M, la masse volumique ρ de la solution et la masse molaire M du soluté. Deux formes équivalentes selon l'unité de ρ :

Si ρ en g·L⁻¹ :
Si ρ en g·mL⁻¹ (notation flacon) :
Ces deux formules sont équivalentes : 1 g·mL⁻¹ = 1000 g·L⁻¹, donc le facteur 10 compense le passage mL→L.
💡 Démonstration :
Dans 1 L de solution, la masse totale est ρ (g·L⁻¹)
La masse de soluté A est : m(A) = %M/100 × ρ
La quantité de matière est : n = m(A)/M = (%M × ρ)/(100 × M)
Et c = n/V = n/1 L → c = (%M × ρ) / (100 × M)

Avec ρ en g·mL⁻¹ : ρ (g·L⁻¹) = ρ (g·mL⁻¹) × 1000
→ c = (%M × ρ × 1000) / (100 × M) = (10 × %M × ρ) / M

✏️ Application — Solution commerciale exprimée en degré (% massique)

Une solution commerciale d'acide sulfurique H₂SO₄ porte l'indication : %M = 96° et ρ = 1,84 g·mL⁻¹.
Données : M(H₂SO₄) = 98,0 g·mol⁻¹

💡 Attention : Sur les flacons de réactifs, le degré (°) exprime le pourcentage massique. Ainsi 96° signifie %M = 96 %. La masse volumique est toujours donnée en g·mL⁻¹ sur les flacons.
c(H₂SO₄) = mol·L⁻¹
%M = 96° → 96 %  |  ρ = 1,84 g·mL⁻¹  |  M = 98,0 g·mol⁻¹
On utilise : c = (10 × %M × ρ) / M
c = (10 × 96 × 1,84) / 98,0 = 1766,4 / 98,0
c ≈ 18,0 mol·L⁻¹

C'est une solution très concentrée et très corrosive — toujours diluer en versant l'acide dans l'eau, jamais l'inverse !

✏️ Application — Solution exprimée en degré (% volumique / titre alcoolique)

Un flacon d'éthanol de laboratoire indique 70°GL (degré Gay-Lussac), ce qui signifie 70 % en volume d'éthanol pur dans la solution.
Données : ρ(éthanol pur) = 0,789 g·mL⁻¹  |  M(éthanol) = 46,0 g·mol⁻¹

💡 Différence clé : Le degré Gay-Lussac (°GL) ou titre alcoolique exprime un pourcentage volumique (%V) : dans 100 mL de solution, il y a 70 mL d'éthanol pur. Ce n'est pas un pourcentage massique ! La concentration molaire se calcule alors différemment.

Calculer la concentration molaire c d'éthanol dans 1 L de solution :

c(éthanol) = mol·L⁻¹
Dans 1 L = 1000 mL de solution à 70°GL : V(éthanol) = 70 % × 1000 = 700 mL d'éthanol pur
m(éthanol) = ρ × V = 0,789 × 700 = 552,3 g
n(éthanol) = m / M = 552,3 / 46,0
c ≈ 12,0 mol·L⁻¹

Méthode générale pour % volumique :
c = (%V/100 × ρsoluté pur × 1000) / M
c = (0,70 × 0,789 × 1000) / 46,0 ≈ 12,0 mol·L⁻¹ ✓

Rendement & écart relatif

Le rendement d'une réaction :

L'écart absolu est la différence absolue entre une valeur expérimentale (ou mesurée) et une valeur théorique (ou de référence).

L'écart relatif permet de donner la précision des mesures par rapport aux valeurs théorique ou de référence :

❓ À quel moment peut-on dire que la valeur expérimentale est juste ?

En pratique au lycée, on considère qu'une mesure est acceptable lorsque l'écart relatif est inférieur à 5 %.

Cependant, ce seuil dépend du contexte expérimental :
• < 1 % → excellente précision (verrerie jaugée, balance de précision)
• 1 à 5 % → précision correcte pour un TP de lycée
• > 5 % → précision insuffisante, il faut chercher les sources d'erreur (erreur de manipulation, réactif impur, réaction incomplète…)

Important : un écart relatif faible ne garantit pas l'exactitude si une erreur systématique est présente (ex. étalonnage incorrect). La valeur est dite juste si elle est proche de la valeur vraie, et fidèle si les mesures sont reproductibles.

📝 Exercice 1 — Masse molaire de l'acide sulfurique H₂SO₄

Données : M(H) = 1 g·mol⁻¹, M(S) = 32 g·mol⁻¹, M(O) = 16 g·mol⁻¹.
Quelle est la masse molaire M de H₂SO₄ ?
A M = 64 g·mol⁻¹
B M = 96 g·mol⁻¹
C M = 98 g·mol⁻¹
D M = 34 g·mol⁻¹
✅ Bonne réponse : C — M(H₂SO₄) = 98 g·mol⁻¹
M = 2 × M(H) + M(S) + 4 × M(O)
M = 2 × 1 + 32 + 4 × 16 = 2 + 32 + 64 = 98 g·mol⁻¹

On additionne les masses molaires atomiques de tous les atomes présents dans la molécule, en tenant compte du nombre de chaque atome donné par les indices dans la formule brute.

📝 Exercice 2 — Quantité de matière d'un solide

On pèse 5,85 g de chlorure de sodium NaCl. Données : M(Na) = 23 g·mol⁻¹, M(Cl) = 35,5 g·mol⁻¹.
Quelle est la quantité de matière de NaCl ?
A n = 0,050 mol
B n = 0,100 mol
C n = 1,00 mol
D n = 0,200 mol
✅ Bonne réponse : B — n = 0,100 mol
M(NaCl) = 23 + 35,5 = 58,5 g·mol⁻¹
n = m / M = 5,85 / 58,5 = 0,100 mol

Vérification dimensionnelle : g ÷ (g·mol⁻¹) = mol ✓

📝 Exercice 3 — Concentration molaire

On dissout 4,00 g de soude NaOH (M = 40,0 g·mol⁻¹) dans de l'eau pour obtenir 500 mL de solution.
Quelle est la concentration molaire de cette solution ?
A c = 0,010 mol·L⁻¹
B c = 0,050 mol·L⁻¹
C c = 0,200 mol·L⁻¹
D c = 8,00 mol·L⁻¹
✅ Bonne réponse : C — c = 0,200 mol·L⁻¹
n(NaOH) = m / M = 4,00 / 40,0 = 0,100 mol
V = 500 mL = 0,500 L
c = n / V = 0,100 / 0,500 = 0,200 mol·L⁻¹

Attention à la conversion mL → L : diviser par 1000 !

📝 Exercice 4 — Dilution

On veut préparer 250 mL d'une solution d'acide chlorhydrique à c₂ = 0,10 mol·L⁻¹ à partir d'une solution mère à c₁ = 2,0 mol·L⁻¹.
Quel volume V₁ de solution mère faut-il prélever ?
A V₁ = 25,0 mL
B V₁ = 5,00 mL
C V₁ = 12,5 mL
D V₁ = 50,0 mL
✅ Bonne réponse : C — V₁ = 12,5 mL
Relation de dilution : c₁ × V₁ = c₂ × V₂
V₁ = (c₂ × V₂) / c₁ = (0,10 × 250) / 2,0 = 12,5 mL

Facteur de dilution : F = c₁/c₂ = 2,0/0,10 = 20. On divise le volume par 20 : 250/20 = 12,5 mL ✓

En pratique, on utilise une pipette jaugée de 12,5 mL et une fiole jaugée de 250 mL.

📝 Exercice 5 — Gaz et volume molaire

À température et pression normales (CNTP), le volume molaire des gaz est Vm = 22,4 L·mol⁻¹.
On dispose de 5,60 L de dioxygène O₂ dans ces conditions.
Quelle est la quantité de matière de O₂ ?
A n = 2,25 mol
B n = 0,250 mol
C n = 0,500 mol
D n = 125 mol
✅ Bonne réponse : B — n = 0,250 mol
n = V / Vm = 5,60 / 22,4 = 0,250 mol

Vérification : L ÷ (L·mol⁻¹) = mol ✓

À 25°C et 1 atm (conditions habituelles de TP), Vm ≈ 24,4 L·mol⁻¹. Toujours vérifier les conditions données dans l'énoncé !

📝 Exercice 6 — Pourcentage massique et concentration molaire

Une solution commerciale d'acide acétique CH₃COOH a un pourcentage massique %M = 8,0 %, une masse volumique ρ = 1 010 g·L⁻¹ et M(CH₃COOH) = 60,0 g·mol⁻¹.
Quelle est la concentration molaire de cette solution ?
A c ≈ 0,80 mol·L⁻¹
B c ≈ 1,35 mol·L⁻¹
C c ≈ 13,5 mol·L⁻¹
D c ≈ 0,135 mol·L⁻¹
✅ Bonne réponse : B — c ≈ 1,35 mol·L⁻¹
c = (%M × ρ) / (100 × M)
c = (8,0 × 1010) / (100 × 60,0) = 8080 / 6000 ≈ 1,35 mol·L⁻¹

Interprétation : dans 1 L de solution (masse 1 010 g), il y a 8,0 % de soluté soit 80,8 g, ce qui donne 80,8/60,0 ≈ 1,35 mol.
